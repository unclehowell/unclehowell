# Openclaw Studio Skills Overview

Openclaw Studio leverages a robust skills system to extend agent capabilities. Skills are modular components that agents can load and execute to perform specific tasks.

## Skill Categories

Skills are categorized to help users discover relevant functionalities:

-   **AI & ML:** For tasks involving machine learning models, data analysis, and AI experimentation.
    -   LLM interaction
    -   Model training and fine-tuning
    -   Vector databases
    -   Evaluation frameworks
-   **Development:** For software development tasks.
    -   Code generation and review
    -   Version control (Git)
    -   Testing and debugging
    -   DevOps and deployment
-   **Productivity:** For enhancing daily workflows and task management.
    -   Document processing (OCR, PDF)
    -   Email management
    -   Calendar and task scheduling
    -   Note-taking
-   **Web & Search:** For accessing and processing information from the internet.
    -   Web scraping
    -   Search engine integration
    -   Content fetching and summarization
-   **Communication:** For interacting with external messaging platforms.
    -   Telegram, Discord, Slack integration
    -   SMS and Email
-   **Creative:** For generating creative content.
    -   Text-to-speech
    -   Image and music generation
    -   ASCII art and video

## Managing Skills

-   **Discovery:** Browse available skills in the Skill Marketplace.
-   **Installation:** Install skills for your agents with a single click.
-   **Configuration:** Configure installed skills through the agent's settings.
-   **Updates:** Keep your skills up-to-date for the latest features and security patches.

## Contributing Skills

We encourage the community to contribute new skills. Please refer to the [Openclaw Contribution Guide](https://github.com/openclaw/openclaw/blob/main/CONTRIBUTING.md) for details on how to develop and submit your own skills.
