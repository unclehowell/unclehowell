# Openclaw Studio UI Guide

This guide provides an overview of the Openclaw Studio user interface and how to navigate its features.

## Dashboard

The main dashboard provides a high-level overview of your agents, their status, and recent activity.

- **Agent List:** Displays all registered agents with their current status (running, stopped, error).
- **Activity Feed:** Shows a log of recent agent actions and communication.
- **Resource Monitor:** (Optional) Displays system resource usage by agents.

## Agent Management

### Creating an Agent
1. Navigate to the 'Agents' section.
2. Click 'Create New Agent'.
3. Fill in the agent's name, description, and select a base model or template.
4. Configure initial skills and memory settings.
5. Save the agent.

### Managing Agents
- **Start/Stop:** Control the execution of individual agents.
- **Configuration:** Edit agent settings, skills, and memory configurations.
- **Logs:** View detailed logs for each agent's activity.
- **Memory:** Inspect and interact with the agent's memory.

## Skill Marketplace

Browse and install available skills to enhance agent capabilities.

- **Search:** Find skills by name or keyword.
- **Categories:** Filter skills by function (e.g., 'Web Search', 'File Management', 'Communication').
- **Install/Uninstall:** Manage skills for your agents.

## Settings

Configure global application settings, including:
- Default LLM provider
- API keys
- Theme and appearance
- Notification preferences

## Help and Support

Access documentation, FAQs, and support channels from the 'Help' menu.
