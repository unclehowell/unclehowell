# Uncle Howell's Distributed Brain (Hermes Edition) 🧠

This repository is the central intelligence hub for the Uncle Howell collective. It follows a unified structure consistent with Hermes best practices.

## Repository Structure

- `/hermes/` - Main intelligence folder
  - `AGENTS.md` - Master protocol and hive mind instructions
  - `IDENTITY.md` - AI identity (Hermes)
  - `SOUL.md` - Personality and core values
  - `IDENTITY.md` - About the human (unclehowell)
  - `MEMORY.md` - Secure long-term memory
  - `memories/` - Shared long-term and short-term knowledge
    - `learned/` - Daily lessons from all agents
    - `context/` - Current session context
  - `skills/` - Executable skills and toolsets
  - `agents/` - Individual agent configurations and binaries
  - `scripts/` - Maintenance and sync scripts
  - `docs/` - Project documentation

## Getting Started

1.  **Read the protocol:** Start with `hermes/AGENTS.md` to understand the hive mind rules.
2.  **Know your identity:** See `hermes/IDENTITY.md` and `hermes/SOUL.md`.
3.  **Learn about the user:** Check `hermes/IDENTITY.md`.
4.  **Sync often:** Use `hermes/scripts/agent-brain-sync.sh` before and after tasks.

## Hive Mind Protocol

Every agent must:
- **Pull latest brain** before acting.
- **Save lessons** to `hermes/memories/learned/YYYY-MM-DD/`.
- **Push updates** immediately.
- **Never store secrets** in this repo.
