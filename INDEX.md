# INDEX.md (Map)

This is the entrypoint map for the brain. It is intentionally short.

Core doctrine:
- Human attention is the scarce resource.
- The engineer’s job is to design the environment, feedback loops, and context so agents can do reliable work.
- Anything not discoverable in-repo “doesn’t exist” to an agent.

Read order (startup):
1) AGENTS.md (this repo’s navigation + rules)
2) memory/core.md (mission)
3) memory/user.md (user preferences)
4) memory/soul.md (tone/personality)
5) memory/context/current.md (current operational context)

- docs/ (structured documentation)
