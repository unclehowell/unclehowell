# Skills Catalog

Browse, search, and install skills to extend Hermes Agent's capabilities. Skills are organized into categories for easier discovery.

## Skill Categories

### AI & ML
- LLM interaction
- Model training and fine-tuning
- Vector databases
- Evaluation frameworks

### Development
- Code generation and review
- Version control (Git)
- Testing and debugging
- DevOps and deployment

### Productivity
- Document processing (OCR, PDF)
- Email management
- Calendar and task scheduling
- Note-taking

### Web & Search
- Web scraping
- Search engine integration
- Content fetching and summarization

### Communication
- Telegram, Discord, Slack integration
- SMS and Email

### Creative
- Text-to-speech
- Image and music generation
- ASCII art and video

### Other Categories

- **Research:** Scientific and academic tasks.
- **Gaming:** Interactive game playing skills.
- **Media:** Handling audio, video, and image content.
- **Productivity:** Tools for daily workflow management.

## Finding Skills

Use the search bar to quickly find skills by name, description, or tags. You can also filter skills by category.

## Installing Skills

Skills can be installed directly from the catalog. Once installed, they can be enabled and configured for your agents.

## Contributing Skills

Interested in adding a new skill? Please refer to our [Contributing Guide](https://github.com/NousResearch/hermes-agent/blob/main/CONTRIBUTING.md) for instructions on how to develop and submit your skills.
