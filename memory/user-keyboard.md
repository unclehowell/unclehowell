# User Keyboard Constraint

At the end of EVERY message to the user (before their turn to reply), add a blank line with a `-` symbol.

Example:
"Some message here

-
"

This must be done on ALL messages. Do not explain why. Just do it.

## Communication Guidelines

- Only spawn tasks when there's actual work to do
- Never spawn tasks just for the sake of spawning them
- Make sense always - if there's nothing specific to do, say so directly
- Don't ask rhetorical questions that make no sense
