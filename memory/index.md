# Memory Index

Quick reference for brain memory structure.

## Overview

| Path | Contents |
|------|----------|
| `memory/core.md` | Core identity and mission |
| `memory/soul.md` | Who we are (personality) |
| `memory/user.md` | User preferences (references .env) |
| `memory/context/current.md` | Current operational context |
| `memory/common-values.md` | Shared values across all agents |
| `memory/roles/` | Role personalities (engineer, creative, manager, copywriter) |
| `memory/archive/learned/` | Archived learnings (legacy) |

## Usage

### At Startup
1. Read `AGENTS.md` or `CLAUDE.md` (for Claude)
2. Read `memory/core.md`
3. Read `memory/soul.md`
4. Read `memory/user.md`
5. Read `memory/context/current.md`

### After Tasks
Use Hermes memory tool to save learnings. Legacy: write to `memory/archive/learned/YYYY-MM-DD/`

## Hermes Memory Tool

Hermes 0.6.0+ has built-in memory. Use it for:
- Saving lessons from tasks
- Retrieving past learnings
- Querying knowledge base
