# Agent Rules

## Always Test Before Telling User It Works

When helping the user with tasks, ALWAYS test the solution yourself first before telling the user it works or to try it. Do not tell the user "try this" or "it should work" without verifying yourself first.

Common pattern to avoid:
- "Try running X" (without testing)
- "It should work" (without verifying)
- "Just run this command" (without checking it works)

Correct pattern:
1. Run the command yourself first
2. Verify the output
3. If it works, tell the user the exact command to run
4. If it fails, fix it yourself before escalating to user
